# Retro Phone Vedio Convert - 复古手机视频效果

Retro Phone Vedio Convert是一个网页工具，可以把普通视频处理成复古风格，模拟老手机拍摄效果，同时支持画面噪声和音频噪声，让视频更有年代感。

## 功能

* 像素化处理，可调节块大小和波动
* 降帧效果，可模拟老设备帧率抖动
* 画面噪声和音频噪声可独立控制
* 暖色、CRT 扫描线、VHS 拉丝效果
* 水平轻微滚动，增强老设备风格
* 支持视频上传、实时预览和录制下载

## 使用方法

1. 打开网页，选择要处理的视频文件
2. 调整各个参数滑条
3. 点击“开始录制并处理”，实时预览处理效果
4. 点击“停止并生成文件”，生成处理后的视频并下载

## 注意事项

* 建议使用 Chrome 或 Edge 浏览器，其他浏览器可能不完全兼容
* 下载的视频可选择格式为 `webm`、`mp4`，大部分播放器可直接播放
* 若出现无声音或播放异常，可尝试降低音频降采样因子

## 技术说明

* 视频处理功能完全在浏览器端完成，无需上传 (可实现离线断网操作，确保视频安全性)
* 使用 `<canvas>` 渲染画面特效
* 使用 `MediaRecorder` 录制处理后的视频和音频
